# MVP_ML
Descrição do dataset: Componentes do PCA que preservam 90% da variância das matrizes de correlação entre resíduos de protéinas de espécies de coronavírus.

O conjunto de dados contém representações obtidas por PCA de matrizes de correlação entre resíduos de proteínas de coronavírus derivadas de simulações de dinâmica molecular de 500 ns. 
Estrutura dos arquivos:
1) O arquivo pca_90_components.csv disponibiliza as coordenadas das amostras (scores) e os rótulos das classes
2) O arquivo pca_90_for_mds.npz fornece os mesmos dados em formato NumPy para análise de MDS com a finalidade de reduzir a dimensionalidade e aprimorar a visualização dos dados.
